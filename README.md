This is the rebuild of [www.thefontains.com](https://www.thefontains.com) using [Next.js](https://nextjs.org), [React](https://react.dev), and [Tailwind CSS v4](https://tailwindcss.com/docs/installation/framework-guides/nextjs).

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the site by modifying `src/app/page.js`. The page auto-updates as you edit files.

## Key Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- ESLint for code quality
- `src/` directory structure enabled
